# Chat Pack

Add a chat application to your Nebari cluster so your team can talk to AI assistants without leaving the platform. The pack ships a chat interface and a customizable agent backend, with Nebari single sign-on built in so only authorized users can access it.

- **Status:** Official (OpenTeams)
- **Maintainer:** [nebari-dev](https://github.com/nebari-dev)
- **Source:** [nebari-dev/nebari-chat-pack](https://github.com/nebari-dev/nebari-chat-pack)

For install and configuration, see the [upstream README](https://github.com/nebari-dev/nebari-chat-pack#readme).
