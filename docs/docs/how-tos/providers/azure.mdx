# Microsoft Azure

:::warning[🚧 Planned, not yet implemented]
Azure scaffolding is more developed than the other planned providers (see [`nebari-infrastructure-core/pkg/provider/azure`](https://github.com/nebari-dev/nebari-infrastructure-core/tree/main/pkg/provider/azure)), but it is not yet supported as a production deploy target.
:::

Community contributions to advance this provider are welcome. Track upstream progress in the [`nebari-infrastructure-core`](https://github.com/nebari-dev/nebari-infrastructure-core) repository.
