# Google Cloud Platform (GCP)

:::warning[🚧 Planned, not yet implemented]
GCP is on the NKP roadmap but is not production-ready today. Stub scaffolding lives in [`nebari-infrastructure-core/pkg/provider/gcp`](https://github.com/nebari-dev/nebari-infrastructure-core/tree/main/pkg/provider/gcp), but provisioning is not wired up end-to-end.
:::

Community contributions to advance this provider are welcome. Track upstream progress in the [`nebari-infrastructure-core`](https://github.com/nebari-dev/nebari-infrastructure-core) repository.
